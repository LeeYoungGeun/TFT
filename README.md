## TFT 커뮤니티 사이트

<br>

### 프로젝트 소개
---
기본적인 CURD를 이용한 커뮤니티 사이트입니다.
다양한 주제로 소통할 수 있습니다.


### 개발인원
### 팀장 이영근
#### 회원기능(회원 가입, 로그인, 마이페이지, 소셜로그인, 이메일 인증(미완성)) 회원 CSS

<br>

### 팀원 김민결 
#### 게시판 CURD / 게시판 CSS  /  이미지 업로드기능(로컬 저장) / 게시글 좋아요 기능.

<br>

### 팀원 이현관
#### 댓글기능  / 댓글 CSS
